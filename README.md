# RevolutionX STEM Racing

Sitio web oficial del equipo RevolutionX del IES José Saramago - Equipo líder de STEM Racing con IA integrada.

## 🤖 Asistente de IA con OpenRouter

### Configuración del API Key

1. **Obtener API Key gratuita**:
   - Visita [OpenRouter](https://openrouter.ai/keys)
   - Crea una cuenta gratuita
   - Genera tu API key

2. **Configurar en la aplicación**:
   - Agrega tu API key al archivo `.env`:
     ```
     VITE_OPENROUTER_API_KEY=tu_api_key_aqui
     ```
   - O configúrala directamente en el asistente usando el botón de configuración

### Modelos Disponibles

- **Llama 3.1 8B** (Gratuito): `meta-llama/llama-3.1-8b-instruct:free`
- **GPT-3.5 Turbo**: `openai/gpt-3.5-turbo`
- **Claude 3 Haiku**: `anthropic/claude-3-haiku`

- ✅ **Respuestas en tiempo real** con streaming
